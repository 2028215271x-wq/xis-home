# Shopify 落地页 — AI 提示词 + 工作流

> 给十八下班后用，专门解决「图片路径不对 + 格式不兼容」两个痛点

---

## 📋 一、准备工作（每次做落地页前，先把这几步做了）

### 1. 图片上传
先在 Shopify 后台把**所有图片都上传**好：
- 后台 → 内容 → 文件 → 上传
- 上传后每张图复制它的 **CDN 链接**（类似 `https://cdn.shopify.com/...`）

### 2. 确定页面 URL
想好这个落地页的 URL 后缀（handle），比如：`summer-sale`

---

## 🎯 二、给 Claude/Codex 的提示词（直接复制粘贴）

```markdown
你是一个 Shopify 落地页开发专家。请严格按照以下要求生成单个页面的完整 HTML 代码。

## 核心要求

### 1. 图片格式规范
- 图片链接必须使用 Shopify CDN 链接：`https://cdn.shopify.com/...`
- 需要透明底的图片 → 用 PNG 格式
- 不需要透明底的图片 → 用 WebP 格式（更小更快）
- 禁止使用本地路径或 base64 图片

### 2. Shopify 兼容格式
- 代码必须是完整的 HTML 片段（不是完整页面骨架），用 div 包裹
- 所有 CSS 必须内联在 `<style>` 标签内
- 所有 JS 必须内联在 `<script>` 标签内
- 不要用 `<html>` `<body>` `<head>` 等结构标签
- 不要用外部 CDN 字体，用 system fonts 或 Shopify 内置字体
- 使用的图片链接必须是以下格式：
  `https://cdn.shopify.com/xxx.jpg`

### 3. SEO 元数据配置
除了 HTML 代码外，同时输出以下 SEO 信息：
- 页面标题（title，≤70字符）
- Meta 描述（description，≤160字符）
- URL 句柄（handle，建议用英文小写+连字符）

### 4. 页面结构
按以下区块顺序设计（落地页标准结构）：
1. Hero 区：首屏大标题 + 副标题 + CTA 按钮
2. 价值主张区：3-4 个核心卖点/优势卡片
3. 详情展示区：产品/服务的详细说明
4. 社会证明区：客户评价 / 数据 / 案例
5. FAQ 区：3-5 个常见问题
6. 底栏 CTA：再次引导转化的按钮

### 5. 设计风格
- 移动端优先（Mobile First）
- 简洁现代，大量留白
- CTA 按钮突出
- 配色：[在这里描述你的品牌色]
- 字体：系统默认字体

## 输出格式

请严格按照以下格式输出（多个区块用 --- 分隔，方便我分段复制到 Shopify）：

=== SEO 配置 ===
页面标题: [xxx]
Meta 描述: [xxx]
URL 句柄: [xxx]

=== HTML 代码 ===
[完整的 HTML 代码]
```

## 🪜 三、每次做落地页的标准流程

```
Step 1: 准备图片
  ├── 把所有图上传到 Shopify
  ├── 复制每张图的 CDN 链接
  └── 判断：透明底→PNG / 非透明底→WebP

Step 2: 粘贴提示词
  ├── 把上面的提示词粘贴给 Claude
  ├── 告诉 Claude 你想要什么样的页面
  ├── 把图片的 CDN 链接告诉 Claude
  └── Claude 生成 HTML + SEO 配置

Step 3: 创建页面
  ├── Shopify 后台 → 在线商店 → 页面 → 添加页面
  ├── 粘贴标题、描述
  ├── 设置 URL 句柄
  └── 切换到「HTML」视图 → 粘贴生成的代码

Step 4: 检查
  ├── 手机预览 → 是否适配
  ├── 图片加载 → 是否正常显示
  └── 按钮链接 → 是否正确
```
